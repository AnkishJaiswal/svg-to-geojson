SVG to GeoJSON
---

Drag and drop SVG on the map and it returns GeoJSON back to you.

### Run it

    git clone https://github.com/mapbox/svg-to-geojson.git

    cd svg-to-geojson
    npm install
    npm start

    open http://localhost:3000/

Runs the app from http://localhost:3000
